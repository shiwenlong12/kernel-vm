Struct page_table::VmFlags

#[repr(transparent)]
pub struct VmFlags<Meta: VmMeta>(_, _);
Expand description

页表项属性。

页表项属性一定完全包含在页表项中，所以独立的页表项属性实现为一个无法获取地址的页表项。
